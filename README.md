# Daheeh-App
Daheeh-App: The Ultimate Study Ecosystem for Tawjihi Students 📱🚀 Built with Flutter, Daheeh is a high-performance productivity platform designed to streamline the study experience with a focus on real-time tracking and offline efficiency.  The application was built on a laptop with an Intel Pentium 3556U processor.


Technical Highlights:

Offline AI OCR: Convert physical notes to digital text 100% offline.

Interactive GPA Simulator: Real-time grade prediction based on the Jordanian system.

Deep Focus Engine: Pomodoro timer with integrated Ambient Soundscapes (Rain, Nature, Library).

Smart Subject Tracking: Dynamic progress bars and syllabus management.

Gamification Logic: XP system, Levels, and Daily Streaks (🔥) to drive consistency.

Advanced UI/UX: Full Dark Mode, custom transitions, and a Gyro-responsive Glow effect.

Live Exam Countdown: Precision tracking for national exam dates.

The app is currently in Private Beta. Would love to hear your feedback on the UI and feature set!

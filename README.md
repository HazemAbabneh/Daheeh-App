# 🚀 Daheeh-App: The Ultimate Study Ecosystem
**Daheeh (دحيح)** is a high-performance productivity platform designed for Jordanian Tawjihi students. Built with Flutter, it bridges the gap between AI technology and academic consistency.

## ✨ Key Features
* **Offline AI OCR:** Convert physical notes to digital text 100% offline.
* **Jordanian GPA Simulator:** Real-time grade prediction tailored to the national system.
* **Deep Focus Engine:** Integrated Pomodoro timer with Ambient Soundscapes.
* **Gamification Logic:** XP system, Levels, and Daily Streaks (🔥) to drive consistency.
* **Advanced UI/UX:** Full Dark Mode with Gyro-responsive Glow effects.

## 🛠 Engineering & Compatibility
* **Development Environment:** Entirely built on an **Intel Pentium 3556U** workstation.
* **Field Testing:** Successfully tested and fully functional on **Samsung Galaxy J5 (J500H)** with only **1.5GB RAM**.
* **System Environment:** Running **Android 10** via **Bootleggers OS** custom ROM.
* **Architecture:** Optimized for ARMv7a and ARMv64-v8a.
